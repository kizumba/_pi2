Progresso do desenvolvimento do Projeto

Framework web em Flask com Banco de Dados SQLite

Todas as entidades do aplicativo possuem as quatro operações básicas do banco de dados: Criação, Leitura, Atualização e Remoção.

Acessibilidade

WCAG 2.1.3 - Teclado (sem exceção) [AAA]

Qualquer ação na aplicação é possível com o teclado.

WCAG 1.4.6 - Contrastes melhorado [AAA]

O usuário consegue aumentar ou diminuir a fonte para melhorar a leitura. Todos os itens do menu estão realçados.

Testes

  • Se a página inicial retorna status code 200
  • Se retorna status code 401 para usuário não autorizado
  • Se o link entrar existe no menu
  • Se a página entrar retorna status code 200
  • Se o usuário consegue cadastrar novo registro
  • Se o usuário consegue se conectar ao sistema
  • Se conectado o link diâmico sair existe
  • Se os botões Cadastrar Usuário, Editar e Exlcuir existem
  • Se o usuáiro consegue acessar a página de editar registro
